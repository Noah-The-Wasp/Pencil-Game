# Pencil-Game
Pencil Game is my second year university game design project

This project is written without many of the code conventions I learned and perfected when making AIMLESS.

For instance, this project relies on a lot of public variables and archaic ways to pass them between scripts. During the development of AIMLESS I learned far more efficient methods for achieving the same results.

Another part of this project which is programmed inefficiently is my reliance on nested if statements which I have also grown out of using.

Pencil Game's variables also do not follow standard naming conventions which I have since implemented into my projects, using camel casing and underscores to make code more readable and easier to navigate.
